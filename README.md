üìÅ Dataset Information
The dataset contains comprehensive avocado sales data with the following attributes:

Date: Transaction date
AveragePrice: Average price of a single avocado
Total Volume: Total number of avocados sold
Size Categories: 4046, 4225, 4770 (PLU codes representing different sizes)
Bag Types: Small Bags, Large Bags, XLarge Bags
Type: Conventional or Organic
Year: Year of sale
Region: Geographic region in the US

USA Regions covered include: Albany, Atlanta, Baltimore-Washington, Boise, Boston, Buffalo-Rochester, California, Charlotte, Chicago, Cincinnati-Dayton, Columbus, Dallas-Fort Worth, and New Mexico.
